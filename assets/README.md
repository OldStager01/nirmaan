# Assets Folder

This folder contains images and screenshots for the project documentation.

## Screenshots

### login-screen.png
Login page screenshot showing the NIRMAAN interface with:
- Government of India themed design
- Username and password fields
- Demo credentials section
- Key features panel
- Security notice

## Usage

To add the login screenshot:
1. Save your login page screenshot as `login-screen.png` in this folder
2. The README.md will automatically display it

## Adding More Screenshots

You can add more screenshots here for:
- Dashboard view
- Sensor data page
- Field management
- Mobile responsive views
- Admin panel
- etc.

Then reference them in the main README.md using:
